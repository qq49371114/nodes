# Guardian V21.0 使用说明

## 版本历史

V21.0 - 8大功能增强版 (2024-02-24)
- 锁定系统时间
- 提醒倒计时
- 违规日志记录
- 学习和游戏模式
- 使用统计系统
- 自动截图监控
- 累计奖励机制
- 数据备份恢复

## 新功能

1. 锁定系统时间 - 防止时间绕过
2. 提醒倒计时 - 5分钟+1分钟提醒
3. 违规日志 - SQLite数据库记录
4. 多模式 - 学习/游戏模式切换
5. 使用统计 - 每日时长统计
6. 自动截图 - 每10分钟截屏
7. 累计奖励 - 完成奖励机制
8. 数据备份 - 云端备份

## 文件结构

C:\ProgramData\Guardian\
- guardian.log
- cached_config.json
- usage_stats.db
- bonus_time.json
- screenshots/ (自动截屏目录)

## 首次使用

1. 将V21.0代码复制到本地
2. 以管理员身份运行 (WMI需要权限)
3. 输入超级密码设置
4. 程序自动创建所有文件

## 使用说明

所有8个新功能已整合到V21.0代码中。
详情请见代码注释。
